How Calculated (ringkas):
Nilai outcome dihitung dari agregasi Progress_i indikator kontributor (lihat Section 4.6). Tidak ada kalkulasi di front-end.
Progress_i = (Realisasi_i - Baseline_i) / (Target_i - Baseline_i) → dibatasi 0..1 \nImpact_k = Σ (w_i * Progress_i) → bobot setara jika bobot tidak ditetapkan

Section 4.6 — National Impact Computation Logic

Rumus & logika kondisional yang mengubah input horizontal/vertical menjadi output komputasi nasional.
Progress_i = (Realisasi_i - Baseline_i) / (Target_i - Baseline_i) → clamp 0..1 \nIF baseline & target → progress \nELSE IF baseline only → before-after \nELSE IF comparator → with-without \nELSE → qualitative support \nImpact_k = Σ (w_i * Progress_i)
